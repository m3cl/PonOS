# ponOS
-
псевдо (а может и нет, т.к. собирается в freestanding окружении) операционная система на C
- включает простые команды (cat, ls, echo, write (для записи в файлы), cd, pwd, whoami (мультипользователя не будет) и чудесный ponosfetch который тут пока-что вырезан)
- персистентная ФС с виртуальным диском, собственно не персистентная она потому что при запуске сама стирает диск (это жертва вайбкодинга)
- невероятно понятный интуитивный интерфейс
- нерабочая виндус-совместимость
п.с. на Swindows её скомпилировать невозможно
